Arrowfunctions

The spread operator is used to expand the items within iterables. For example
array1 = [1,3,4] 
array2 = [5,7]
myString = "hello"
sum = [...array1, ...array2, ...string]
expands to [1,3,4,5,7,'h','e','l','l','o']
